One-Time Google Setup
Takes ~3 minutes. Only needed once.
1
Go to Google Cloud Console
Open console.cloud.google.com → Create a new project (or select existing).
2
Enable Gmail API
Go to APIs & Services → Library → Search "Gmail API" → Click Enable.
3
Create OAuth 2.0 Credentials
Go to APIs & Services → Credentials → Create Credentials → OAuth client ID → Choose Web application.

Under Authorised JavaScript origins add:
—
4
Configure OAuth Consent Screen
Go to OAuth consent screen → Set to External → Fill app name → Under Scopes add gmail.send → Add your email as a test user.
5
Copy your Client ID
After creating, copy the Client ID (looks like 1234567890-abc...apps.googleusercontent.com) and paste it below.
Paste the Client ID from step 5 above.
🔐
Sign in with Google
Click below to authorise your Gmail account. A Google sign-in popup will open.

Only gmail.send permission is requested. Emails are sent directly from your account. No data is stored anywhere.
